Structure of the HDL

Structure of VHDL

  • HDL modules follow the general structure of software languages such as c
  • The VHDL module has two major constucts,entity and architecture
  • Entity declares the input & output signals of the system to be described & is given a name or identifier by the user
  • VHDL is case insensitive i.e HALF-ADDER & half-adder are treated as same
  • The name should start with an alphabetical letter & can include the character underscore(_)
  • The inputs & outputs are called input ports and output ports.The name of the port is user selected. 
    • entity half_adder is 
port(I1 : in bit;I2 :in bit;sum_h :out bit;
carry_out:out bit);
end half_adder;
  • In the above entity the names of the Input (signals) ports are I1 &I2
       ":" a Colon instantiates the mode of the port as an input.
    the type of the input signals is bit ,which allows the signals to take either logic 0 or logic 1.
  • The sum_h & carry_out are declared as outputs and their types are bit
  • The order that input & output are written inside the parenthesis is irrelevant
  • ";" semicolon is used as an seperator
  • The architecture includes details about the relationship between inputs and outputs of the system & must be bound to an entity
  • More than one architecture can be bound to the same entity ,but each architecture can be bound to only one entity.
  • Binding here means that information listed in the entity is visible to the architecture

Structure of verilog Module

  • verilog module has a declaration & a body.In declaration name,inputs & outputs of the module are listed
  • The body shows the relationship between the inputs & the outputs.

Example of the verilog Module

module half_adder (I1,I2,O1,O2);
input I1;
input I2;
Output O1;
output O2;
//blank lines are allowed 
assign O1=I1^I2;
assign O2=I1&I2;
endmodule


  • verilog is case sensitive
  • The name of the module should start with an alphabetical letter & can include special character underscore(_)
  • The declaration of the module starts with the predefined word module followed by user selected name.
  • the name of the input & outputs are written inside parenthesis & are separated by commas
  • The closing parenthesis is followed by a semicolon
  • the semicolon ; it is a line separator
  • In verilog module more than one i/p or output can be written on the same line by using ','
