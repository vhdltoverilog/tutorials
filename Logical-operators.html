Logical operators of the HDL

These operators perform logical operational such as AND,OR,NAND,NOR,NOT,EX-OR.

The operation can be on two operands or on a single operand.The operands can be single bit or multiple bits.

VHDL logical operators

These operators should appear only on the right hand side of statements. All VHDL operators are bitwise they operate on the corresponding bits of two signals.

operators operand type Result type
AND bit bit
OR bit bit
NAND bit bit
NOR bit bit
XOR bit bit
XNOR bit bit
NOT bit bit

Verilog logical operators

  • These operators perform logical operations such as AND,OR,EX-OR
  • The verilog logical operators can be classified into three groups bitwise,boolean logical & Reduction.
  • The bitwise operators operate on the corresponding bits of two operands.
  • The boolean logical operators operate on two operands the result is boolean 0 false or 1 true.

verilog bitwise logical operators

operators operand type Result type
& AND bit bit
| OR bit bit
~& NAND bit bit
~| NOR bit bit
^ XOR bit bit
~^ XNOR bit bit
~NOT bit bit

verilog boolean logical operators

operators operation number of operand
&& AND two
|| OR two

verilog Reduction logical operators

Reduction operators operate on a single operand .The result is boolean

operator operation number of operand
& reduction AND one
| reduction OR one
~& reduction NAND one
~| reduction NOR one
^ reduction XOR one
~^ reduction XNOR one
! Negation one
