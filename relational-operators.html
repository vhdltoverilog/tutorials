Relational operators

Relational Operators are implemented to compare the values of two objects.The results is boolean.

VHDL relational operators

operators description operand type result type
= Equality Any type boolean
/= Inequality Any type boolean
< less than scalar boolean
<= less than or equal scalar boolean
> greater than scalar boolean
>= greater than or equal scalar boolean

verilog relational operators

operators description result type
== Equality 0,1,x
!= inequality 0,1,x
=== Equality inclusive 0,1
== Inequality inclusive 0,1
< less than 0,1,x
<= less than or equal 0,1,x
> greater than 0,1,x
>= greater than or equal 0,1,x
